# Notas-de-IA

> Este repositorio es una colección de proyectos y ejercicios prácticos sobre la materia de **Inteligencia Artificial (IA)** . Todos los proyectos están implementados principalmente usando **Google Colab** 

## Contenido del Repositorio: Proyectos de IA

Cada carpeta contiene un proyecto o notebook de Google Colab 

| Carpeta / Proyecto | Tópico Principal | Descripción Breve |
| :--- | :--- | :--- |
| **`1_Flor_Iris`** | Clasificación Clásica | Ejemplo fundamental de clasificación utilizando el famoso *dataset* de la flor Iris. |
| **`1_Neuronal_Multicapa_XOR`** | Redes Neuronales Básicas | Implementación de una red neuronal multicapa para resolver el problema de la compuerta lógica XOR. |
| **`Accion_de_Amazon`** | Series de Tiempo / Predicción | Probable predicción o análisis del precio de las acciones de Amazon. |
| **`Clasificacion_de_Ropa`** | Visión Artificial | Clasificación de imágenes, probablemente utilizando el *dataset* Fashion MNIST. |
| **`Clasificacion_de_digitos`** | Visión Artificial / MNIST | Clasificación de dígitos escritos a mano (típicamente usando el *dataset* MNIST). |
| **`Clasificacion_de_un_candidato_a_un_empleo`** | Clasificación / Negocios | Modelo para predecir si un candidato es apto o no para un puesto de trabajo. |
| **`Perceptron_OR_y_NOT`** | Modelos Clásicos | Implementación del algoritmo Perceptrón para resolver las compuertas lógicas OR y NOT. |
| **`Predecir_el_precio_de_Bitcoin`** | Series de Tiempo / Predicción | Modelo de Machine Learning enfocado en predecir el valor futuro del Bitcoin. |
| **`REDESNEURONALES`** | Fundamentos de Deep Learning | Carpeta que probablemente contiene material introductorio sobre el concepto y la estructura de las Redes Neuronales. |
| **`predicción_de_la_eficiencia_del_combustible`** | Regresión | Modelo para predecir el consumo o la eficiencia del combustible de vehículos (Regresión). |
| **`predicción_de_popularidad_de_una_canción`** | Regresión / Clasificación | Modelo que intenta predecir qué tan popular será una canción en base a sus características. |

## Herramientas Utilizadas

* **Google Colab**: Entorno principal para la creación y ejecución de los notebooks de Python.

##  Cómo Explorar los Proyectos

Dado que la mayoría de los proyectos fueron creados en Google Colab:

1.  Navega a la carpeta del proyecto que deseas revisar (ej. `Predecir_el_precio_de_Bitcoin`).
2.  Busca el archivo del notebook (`.ipynb`).
3.  Haz clic en el archivo y luego en el botón "Open in Colab"  para ejecutarlo directamente en tu navegador.
