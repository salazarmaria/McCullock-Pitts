## Redes Neuronales

| Abreviatura | Nombre Completo | Mejor Uso |
|-------------|------------------|-----------|
| **MLP** | Multilayer Perceptron (Perceptrón Multicapa) | Datos tabulares, clasificación básica, predicción numérica |
| **CNN** | Convolutional Neural Network (Red Neuronal Convolucional) | Imágenes, visión artificial, reconocimiento de objetos y patrones visuales |
| **RNN** | Recurrent Neural Network (Red Neuronal Recurrente) | Texto, lenguaje natural, series de tiempo, datos secuenciales |

> Este repositorio documenta mi proceso de aprendizaje en Inteligencia Artificial, aplicando modelos reales y analizando distintos tipos de datos.
Cada carpeta contiene su cuaderno de Colab con gráficas, código y comentarios.
